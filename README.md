## Features ✨
Minimalist Aesthetic: Squircles, soft glow, and refined typography.

Library Management: Custom folder management for your local music collection.

Background Playback: Robust foreground service support so your music never stops.

Battery Aware: Automatic "low-power" mode when your battery dips below 5%.

Fluid Visualizer: Real-time wave rendering.

Installation 🛠️
Requirements:

Android device

Python 3.x

Kivy & Buildozer

# Clone the repo

```
git clone https://github.com/raind4sher/tsuki.git
cd tsuki
```
# Build the APK

```
buildozer android debug deploy run
(Check the Buildozer documentation for environment setup).
```

### Troubleshooting 🔧
The music stops when the screen turns off?
Make sure the Foreground Service permissions are correctly set in the buildozer.spec file.

App crashes on startup?
Ensure you have granted the necessary Storage Access Framework (SAF) permissions to the app.
